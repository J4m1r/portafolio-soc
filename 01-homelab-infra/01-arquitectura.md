# 01 вҖ” Arquitectura del Lab

## VisiГіn general

El lab estГЎ compuesto por dos hosts fГӯsicos en la misma red local. singularity
corre Proxmox VE y aloja los servicios Linux. CHECO corre Windows 11 con
VirtualBox para el lab de Active Directory y WSL2 para IA local.

## Hardware

### singularity вҖ” Proxmox VE

| Recurso | Detalle |
|---|---|
| CPU | Intel Core i7-1165G7 @ 2.80GHz вҖ” 4 cores / 8 threads |
| RAM | 15 GB |
| Disco principal | NVMe WD SN740 512 GB |
| Disco backup | HDD Seagate 500 GB вҖ” montado en `/mnt/backup` |
| IP | 192.168.1.230 |

### CHECO вҖ” VirtualBox + WSL2

| Recurso | Detalle |
|---|---|
| CPU | Intel Core i5-10400 @ 2.90GHz вҖ” 6 cores / 12 threads |
| RAM | 32 GB |
| GPU | Nvidia 6 GB VRAM |
| SO | Windows 11 Pro |
| IP | 192.168.1.2 |
| Hipervisor | VirtualBox вҖ” VMs en D:\SERVIDORES\ |
