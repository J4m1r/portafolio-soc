# Hola, soy Jamir Marez 
### Administrador de Infraestructura Ssr. en transición a Ciberseguridad (SOC)

Llevo **+8 años** administrando entornos **Windows/Linux**: Active Directory, GPO, redes, virtualización y monitoreo de sistemas críticos. Hoy estoy dando el paso a **ciberseguridad**, no para empezar de cero, sino para poner todo eso al servicio de **proteger lo que antes administraba**.

Esa trayectoria es mi mayor ventaja: después de años sosteniendo infraestructura, **sé cómo se comporta un sistema en condiciones normales**, y esa línea base es justo lo que un analista necesita para distinguir lo anómalo —el acceso que no cierra, el log que no encaja, el proceso que no debería estar ahí—. La detección y la respuesta a incidentes empiezan por entender la operación desde adentro.

Actualmente me capacito de forma activa y aplicada:
- **SIEM / detección:** despliegue de **Wazuh** y análisis de **Windows Event Logs**.
- **Certificación:** cursando **SC-200 (Security Operations Analyst)**, sobre la base de **AZ-900**.
- **Práctica:** laboratorios propios donde rompo, observo y documento.

Este repositorio es la bitácora de ese proceso. Acá vas a encontrar mis proyectos, configuraciones, resoluciones y notas técnicas: **documento cada avance** porque para mí saber hacer algo y poder explicarlo son la misma cosa.

Busco sumarme a un equipo de seguridad donde **aportar desde el primer día** con mi base técnica y seguir creciendo. Si te interesa lo que estoy construyendo, mirá los proyectos de abajo. 👇

Actualmente construyo experiencia práctica en SOC a través de laboratorios, proyectos reales y certificaciones enfocadas en el stack de seguridad de Microsoft.
